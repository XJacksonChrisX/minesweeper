# 🎮 踩地雷遊戲專題

一個使用 HTML、CSS 和 JavaScript 開發的經典踩地雷遊戲。

## 📋 功能特性

### 核心遊戲功能
- ✅ 完整的踩地雷遊戲邏輯
- ✅ 左鍵開啟格子，右鍵標記地雷
- ✅ 自動展開空白區域（周圍無地雷）
- ✅ 計數顯示相鄰地雷數
- ✅ 遊戲計時器

### 遊戲難度
1. **簡單** - 8×8 棋盤，10 個地雷
2. **中等** - 10×10 棋盤，30 個地雷
3. **困難** - 12×12 棋盤，50 個地雷

### 遊戲界面
- 🎨 美觀的現代化設計
- 📱 響應式布局，適配各種螢幕大小
- 🎯 直觀的遊戲操作
- 📊 實時地雷計數和計時

## 🎮 遊戲規則

1. **開啟格子**：左鍵點擊未開啟的格子
   - 如果格子是安全的，會顯示周圍地雷數
   - 如果周圍無地雷，會自動展開相鄰區域
   - 如果踩到地雷，遊戲結束

2. **標記地雷**：右鍵點擊格子進行標記
   - 標記後的格子顯示旗子 🚩
   - 再次右鍵可取消標記

3. **獲勝條件**：安全開啟所有非地雷的格子

4. **失敗條件**：踩到地雷

## 🚀 使用方法

1. 在瀏覽器中打開 `index.html` 文件
2. 選擇難度級別（簡單、中等、困難）
3. 使用以下操作進行遊戲：
   - **左鍵**：開啟格子
   - **右鍵**：標記/取消標記地雷
   - **新遊戲按鈕**：重新開始當前難度的遊戲

## 📚 代碼組織

### 遊戲狀態管理
```javascript
gameState = {
    board: [],        // 棋盤數據
    revealed: [],     // 開啟狀態
    flagged: [],      // 標記狀態
    gameOver: false,  // 遊戲結束
    won: false        // 是否獲勝
}
```

### 核心函數
- `initGame()` - 遊戲初始化
- `placeMines()` - 地雷放置
- `calculateNumbers()` - 計算地雷數
- `revealCell()` - 開啟格子
- `revealEmpty()` - 遞迴展開空白
- `toggleFlag()` - 標記地雷
- `checkWin()` - 檢查獲勝

## 👨‍💻 開發筆記

### 算法重點

1. **地雷放置**：使用隨機數確保地雷均勻分佈
2. **空白展開**：BFS/DFS 遞迴展開相鄰未開啟格子
3. **勝利判定**：檢查已開啟格子數是否等於非地雷總數

### 性能優化

- 使用一維數組代替二維數組（節省記憶體）
- 用索引計算行列坐標
- 事件委託優化事件監聽
