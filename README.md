# 踩地雷 Minesweeper（C++ 簡易版）

一個用 C++ 寫成的主控台踩地雷小遊戲，程式碼集中在單一檔案、約 130 行，適合課堂報告與教學示範。

## 檔案說明

| 檔案 | 說明 |
|------|------|
| `minesweeper.cpp` | 遊戲原始碼（單一檔案，含中文註解） |
| `minesweeper.exe` | Windows 編譯好的可執行檔，可直接執行 |
| `踩地雷報告.pptx` | 介紹這個遊戲與程式的簡報（7 頁） |

## 遊戲規則

- 盤面大小 9 × 9，隨機藏有 10 顆地雷。
- 輸入「列 行」座標（例如 `3 4`）翻開格子。
- 翻到的數字＝該格周圍 8 格的地雷數量。
- 翻到空白（周圍沒雷）會自動往外擴散一整片。
- 踩到地雷 → 遊戲結束；翻完所有安全格 → 獲勝。

## 編譯與執行

需要安裝 C++ 編譯器（例如 MinGW-w64 提供的 `g++`）。

```bash
g++ minesweeper.cpp -o minesweeper.exe
./minesweeper.exe
```

Windows 也可以直接雙擊 `minesweeper.exe` 執行。

## 程式重點

- 用三個二維陣列記錄遊戲狀態：`board`（玩家看到的盤面）、`mine`（地雷位置）、`open`（是否翻開）。
- 用迴圈與條件判斷處理遊戲流程。
- 用**遞迴**實作「翻到空白自動擴散」的效果（`reveal` 函式）。
